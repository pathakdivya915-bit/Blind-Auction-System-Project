# 🏷️ Blind Auction System (Python)

A command-line based Blind Auction application built using Python. This project simulates a real-world sealed-bid auction where participants submit bids privately, and the highest bidder wins without knowing others' bids.

---

## 🚀 Features

* Multiple participants can place bids
* Bids remain confidential (blind auction concept)
* Automatic winner determination
* Input validation for smooth user experience
* Clean and modular code structure

---

## 🛠️ Tech Stack

* **Language:** Python 3
* **Libraries Used:** None (pure Python)


2. Navigate to the project folder:

   ```bash
   cd blind-auction
   ```

3. Run the program:

   ```bash
   python auction.py
   ```

---

## 🎮 How It Works

1. Each participant enters their name and bid amount
2. After each entry, the screen clears to keep bids hidden
3. Continue adding bidders until no participants remain
4. The program calculates and displays the highest bidder
